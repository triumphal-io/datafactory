# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- AI-powered spreadsheet platform with Django backend and React frontend
- Workbook system with sheets, file library, and AI assistant
- AI assistant with tool-calling support (spreadsheet manipulation, web search, file queries)
- Bulk cell enrichment with background job processing
- File upload and processing pipeline (CSV, XLSX, PDF, DOCX, PPTX, TXT, MD)
- RAG-based document querying with ChromaDB vector embeddings
- Web search and scraping via Playwright
- Real-time updates via WebSocket (Django Channels)
- MCP server integration for external AI tools
- Multi-provider LLM support via LiteLLM (OpenAI, Anthropic, Google)
- Configurable AI model per workbook
- Provider credential management
